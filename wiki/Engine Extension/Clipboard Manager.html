Clipboard Manager

Description:

Clipboard Manager keeps track of the clipboard with embedded collection.


Info:

  • Path: {src}/EngineExtensions/ClipboardManager.js
  • Exports: Class ClipboardManager()
  • Responsible to initialize: EngineExtensions
  • API access: Engine.ClipboardManager

API Exported:

  1. push(item): Push information to clipboard and also on collection.
  2. top(): Returns the clipboard item.
  3. getCollection(): Returns all the items on the collection.
  4. copyObject(obj): Copy Object to clipboard.
  5. pasteObject(): Create a new Object based on the copied object on clipboard.
