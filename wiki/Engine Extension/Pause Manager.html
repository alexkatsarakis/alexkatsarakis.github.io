Pause Manager

Description:

Pause Manager is responsible to pause and resume the game and the animations.


Info:

  • Path: {src}/EngineExtensions/PauseManager.js
  • Exports: Class PauseManager()
  • Responsible to initialize: EngineExtensions
  • API access: Engine.PauseManager

API Exported:

  1. pause(): Pause most of the actions happening except input and rendering.
  2. resume(): Resume the game and resume animations.
