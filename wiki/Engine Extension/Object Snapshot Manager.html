Object Snapshot Manager

Description:

Object Snapshot Manager is responsible to keep a collection of Snapshots of objects.


Info:

  • Path: {src}/EngineExtensions/ObjectSnapshotManager.js
  • Exports: Class ObjectSnapshotManager()
  • Responsible to initialize: EngineExtensions
  • API access: Engine.ObjectSnapshotManager

API Exported:

  1. snapshotObject(obj): Save a snapshot of the object.
  2. getSnapshots(obj): Returns all the saved snapshots for the given object.
  3. getAllSnapshots(): Returns all the saved snapshots.
  4. resetObjectToSnapshot(objID, snapID): Set objects state to the snapshot state.
