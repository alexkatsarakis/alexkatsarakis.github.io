Animation Workshop

Description:

Animation Workshop is used from the user in order to create new animations based on films that are installed in Animation Manager (see Animation Manager tab for more details).


How does it work?

  1. The user clicks to open the tab then all the installed films have a visual representation through a box on the screen.
  2. The user picks one film that will be played on the animation. (by clicking on the corresponding box)
  3. When picked a popup window will come on the screen and will provide some information to the user.
    • A panel that shows each individual frame of the film.
    • A panel that shows a live representation of how the animation will look like when created.
    • A panel to configure the animation delay, dx, dy, and id (name)
  4. When created, the animation will be available to be used by the user through code.
