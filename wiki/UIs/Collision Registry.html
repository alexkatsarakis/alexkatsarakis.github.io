Collision Registry

Description:

Collision Registry is a tab where the user is able to register a new collision between to objects.


How does it work?

When the collision registry button is pressed there is a UI revealed that shows all the current registered collision between objects. And there is a box where a user can create a new collision between two objects. Afterwards, the user is able to code the behavior on collision object (as an event).


Future Plans:

When the Collision Manager (check collision manager tab) is expanded with surface the UI will be expanded to serve this feature.
