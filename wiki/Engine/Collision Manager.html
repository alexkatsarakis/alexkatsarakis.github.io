Collision Manager

Description:

Collision Manager is responsible to keep the collisions, check and fire collision events.


Info:

  • Path: {src}/Engine/collisions/collisions.js
  • Exports: Class CollisionManager()
  • Responsible to initialize: Engine
  • API access: Engine.CollisionManager

API Exported:

  1. installCollision(obj1ID, obj2ID, code): Adding a new Collision to the manager between two objects.
  2. removeCollision(obj1ID, obj2ID): Removing the collision.
  3. getAllCollisions(): Returns an object with all the collisions.
  4. getCollisions(objID): Returns an object with all collision involving the object with the given id.
  5. getCollision(obj1ID, obj2ID): Returns the collision (if it exists) of two objects.
  6. setCollision(obj1ID, obj2ID, code): Setting the code to be executed on collision of two objects.

Future Plans:

Change general collisions to surface collision. Meaning you will be able to add collision between an object and side of an object.
