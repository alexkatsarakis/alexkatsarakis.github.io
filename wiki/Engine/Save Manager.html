Save Manager

Description:

Save Manager is responsible to save/retrieve the game state either to remote Database or locally from JSON file if DB is not available.


Info:

  • Path: {src}/Engine/save/save.js
  • Exports: Class SaveManager()
  • Responsible to initialize: Engine
  • API access: Engine.SaveManager

API Exported:

  1. setEngine(): This function is crucial in order to load the game because it loads the information needed. Must be called before Engine.start().
  2. saveObjects(): Will save the objects to Database if you are database is available or download a JSON file
  3. getObjects(): Will retrieve the objects either from Database or from the JSON file

Future Plans:

  1. saveGameState(): Will save all the game state
  2. getGameState(): Will retrieve the whole game state
