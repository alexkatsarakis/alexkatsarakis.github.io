Input Manager

Description:

Input Manager is responsible to handle anything related to input. That includes keys, key combinations, controller.


Info:

  • Path: {src}/Engine/input/InputManager.js
  • Exports: InputManager inputManager
  • Responsible to initialize: InputManager.js
  • API access: Engine.InputManager

API Exported:

  1. keyPressed(key, keepFiring): Mark the key as pressed and choose if you want to keep adding it on pressed on each getPressedKeys() call.
  2. keyReleased(key): Mark the key as released and it will return it as released key on the next getReleasedKeys().
  3. getReleasedKeys(): Returns an array of all the keys released since the function was last called.
  4. getPressedKeys(): Returns an array of all the pressed keys since last checked (also includes the keys that are marked as keep keepFiring).
  5. addCombo(id, keys): Gets an id and an array of keys and from now on if these keys are being pressed at the same time this combo will be returned as pressed.
  6. removeCombo(id): Removes the combination with the given id.
