Sound Manager

Description:

Sound Manager is responsible to play sounds and background music and everything associated with sound.


Info:

  • Path: {src}/Engine/sound/sound.js
  • Exports: Class SoundManager()
  • Responsible to initialize: Engine
  • API access: Engine.SoundManager

API Exported:

  1. addSound(tag, path): Install a new sound to the soundManager
  2. removeSound(tag): Remove a sound from the manager
  3. getSounds(): Return all the installed sounds
  4. playSoundOnRepeat(tag): Plays a sound on repeat until it is stopped by either stopSoundOnRepeat or stopAllSounds
  5. stopSoundOnRepeat(tag): Stop a sound that is playing on repeat
  6. playSound(tag): Play an installed sound. (if this sound is already playing it will restart)
  7. stopSound(tag): Stop a sound that is playing.
  8. stopAllSounds(): Stops all the sounds currently playing.

Future Plans:

  1. haveSound(tag): Return true/false if there is a sound installed with this tag
