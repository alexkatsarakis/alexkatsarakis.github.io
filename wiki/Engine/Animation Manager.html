Animation Manager

Description:

Animation Manager is responsible for keeping, creating and manipulating anything related to animations during the game.


Info:

  • Path: {src}/Engine/animations/animations.js
  • Exports: Class AnimationManager()
  • Responsible to initialize: Engine
  • API access: Engine.AnimationManager

API Exported:

  1. getAnimationCategory(catID)
  2. getAnimatorCategory(catID)
  3. progress(gameTime)
  4. timeShift(gameTime)
  5. getAllFilms()
  6. getFilm(filmID)
  7. getAnimation(animID)
  8. getAllAnimations()
  9. registerNewAnimation(anim,filmID)
  10. playAnimation({object,anim,onStart,onFinish,animator})

Currently supported Animation/Animator categories

  • Moving Animator
  • FrameRange Animator
