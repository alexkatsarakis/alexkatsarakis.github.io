Value Manipulation

What is a Value?

Value is something stored on an object and can be used for a lot of reasons.
e.g.
on object 'Score' save an attribute with name 'currentScore' that will save the current Score
on the same object you can save an attribute named 'text' that will save the current text shown to the user.

How Am I able to change a Value?

There are 3 ways to manipulate an object's value

  1. Through code
  2. With Object Information UI
  3. With actions (like drag) when possible
