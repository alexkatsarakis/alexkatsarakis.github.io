What is a script?

A script is a small amount of commands that will be executed on a particular time in order to achieve a goal.


Where do I need to add a script?

Each object have different hook points that you are able to add a script in order to achieve a certain goal. For example, you can add a script that will fire when the game begin, another that will fire each time a value changes, etc.
Of course there are some special events that will be disguised on the appropriate tab.


How does the script interact with the game?

Your code will interact with the game using the games API.
If you are using a Visual Programming Language then there will be blocks to guide you how to structure the script in order to achieve the functionality.
If you are using a Text based language then you will need to read the API.
